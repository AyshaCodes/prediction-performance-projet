# Application de Prédiction de Performance de Projet
🚀 Démonstration en ligne
Retrouvez l'application interactive ici :
https://prediction-performance-projet-kmzwt5qwkbyqvr2zsdm8ec.streamlit.app/

## Description
Application intelligente qui prédit la performance d'un projet (Réussi / En retard / Échoué)
en utilisant un modèle de Machine Learning (Random Forest).

## Technologies utilisées
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- ipywidgets (interface interactive)

## Dataset
Dataset simulé de 300 projets avec les variables suivantes :
- Budget prévu
- Durée prévue
- Taille de l'équipe
- Complexité du projet
- Nombre de risques
- Expérience du chef de projet

## Variable cible
`performance` : Réussi / En retard / Échoué

## Résultats du modèle
- Algorithme : Random Forest Classifier
- Accuracy : ~100%

## Comment exécuter
1. Ouvrir le notebook dans Google Colab
2. Exécuter toutes les cellules
3. Utiliser le formulaire interactif pour prédire
